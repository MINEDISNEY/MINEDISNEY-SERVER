Minedisney Map installation

Instructions

  1. Download the map and resource pack (links below).
  2. Unzip the files. (data is zip because we have around 2000 files)
  3. Place the map folder into your Minecraft saves folder.
  4. Place the resource pack into resourcepacks.
  5. Launch Minecraft, select the resource pack in settings, and enjoy! (1.20 I think) 💪
